# Prompt History

## 1. Generate Program Min-Heap dan Max-Heap (Java dan Python)

Buat program (Java dan Python) menggunakan struktur data Heap untuk menyimpan sejumlah data barang (id dan nama). Program memiliki menu sebagai berikut:
1. Insert data (Tambah data barang ke dalam Min-Heap dan Max-Heap)
2. Display ascending (Tampilkan data terurut ascending dari Min-Heap)
3. Display descending (Tampilkan data terurut descending dari Max-Heap)
4. Delete from Min-Heap (Hapus elemen dengan ID tertentu dari Min-Heap)
5. Delete from Max-Heap (Hapus elemen dengan ID tertentu dari Max-Heap)
6. Load from CSV (Tambah data dari file data.csv ke dalam kedua heap)
7. Exit

NB:
1. Data disimpan dalam dua struktur heap sekaligus (Min-Heap dan Max-Heap)
2. Min-Heap property: parent ≤ children (elemen terkecil di root)
3. Max-Heap property: parent ≥ children (elemen terbesar di root)
4. Operasi insert menggunakan heapifyUp (O(log n))
5. Operasi delete menggunakan heapifyDown (O(log n))
6. Operasi display sorted menggunakan ekstraksi elemen (O(n log n))
7. File CSV berisi data dengan format: ID, Nama
